# TapTap 试玩说明

本说明用于预约页、试玩页、测试招募或后台版本说明。

## 当前测试定位

- 页面阶段：预约 / 试玩准备
- 版本定位：单机动作幸存者原型
- 平台顺序：Android 首发，Windows 随后开放

## 产品说明

`夜幕幸存者` 是一款 2D 幸存者动作游戏。玩家需要在不断增强的怪潮中移动、拾取、升级与进化武器，通过构筑不同能力组合撑过整场战斗。

## 当前版本内容

- 自动攻击与走位生存
- 升级三选一
- 宝箱进化
- 永久升级与本地成长
- 成就与本地排行榜
- 简中、繁中、英文、日文、韩文本地化基础

## 当前版本限制

- 当前仍处于预约 / 试玩准备阶段
- 不包含账号系统
- 不包含联机功能
- 不包含内购功能
- 最终名称、美术包装、部分数值和平衡仍可能继续调整

## 玩家须知

- 推荐优先体验 Android 版本
- Windows 版本将在后续阶段补充开放
- 当前版本以单机游玩为主，进度与设置保存在本地
- 如遇到异常、黑屏、崩溃或文本问题，可通过邮箱反馈：`yurisachan16@gmail.com`
