# 自动游玩代理——游戏设计方案

> 调研参考：Vampire Survivors 的半放置设计哲学、Arknights 录像回放系统、AFK Arena 的离线托管模式、街机时代的 Attract Mode 传统。
> 本文档为纯游戏设计方案，不涉及技术实现细节。

---

## 方案 A：轮回见证者（标题演示模式）

### 设计理念

> 来源于街机时代的 Attract Mode 传统——当游戏无人操作时，自动开始演示以"吸引旁观者"。
> 现代意义：让玩家在标题界面就能感受到游戏的视觉震撼，并随时无缝接管。

### 玩家体验流程

```
标题界面静止 30 秒
    → 屏幕边缘出现"轮回见证中…"字样（半透明，不干扰画面）
    → AI 自动开始一局游戏
    → 玩家可全程旁观，随时按任意键"接管"并切换为正式游玩
    → AI 被击败或通关后，自动返回标题界面
```

### 核心设计要素

**接管机制**
- 玩家旁观时，屏幕右下角常驻"按任意键接管"提示
- 接管后，当前 AI 的局面状态（位置、HP、武器）**原样交给玩家**继续
- 接管的一局不计入排行榜，标记为"见证接管局"

**AI 行为风格**
- AI 展示多种武器组合，每次演示局优先使用不同的 Build，让玩家看到游戏内容的广度
- AI 会犯"人性化的小错误"（偶尔走进敌群、升级时短暂犹豫），避免表演式的完美操作让玩家望而却步
- 遭遇 Boss 时，AI 会明显调整走位策略，向观战玩家"展示"Boss 的危险性

**见证成就**
- 累计旁观 AI 游玩超过 10 分钟 → 解锁称号"轮回旁观者"
- 在 AI 濒死时接管并存活超过 60 秒 → 解锁称号"最后的守护"
- 接管后完成通关 → 解锁隐藏成就"见证者的意志"

### 设计优缺点

| 优点 | 缺点 |
|------|------|
| 玩家打开游戏就能看到精彩画面，降低上手门槛 | AI 表现不够好时反而劝退 |
| 旁观全程是低压力的娱乐体验 | 玩家可能长时间旁观而不主动游玩 |
| 接管机制创造独特的"中途救援"乐趣 | 接管局不计成绩可能让玩家困惑 |

## 方案 B：传承之魂（录像回放 + 亡灵竞速）

### 设计理念

> 来源于 Arknights 的"录像回放"系统——最好的自动游玩不是 AI 决策，而是**将玩家自己的决策重新演绎**。
> 你在看的不是"一个机器人"，而是"过去最强的自己"。

### 玩家体验流程

```
玩家完成一局游戏（无论胜负）
    → 结算界面出现选项："是否将此局存为【传承之魂】？"
    → 存储后，下次游玩时可选择开启"传承模式"

传承模式开启后：
    → 屏幕上出现半透明的"幽灵玩家"，重演上一局的完整路径
    → 幽灵与本局同步推进（第 30 秒时幽灵做了什么，当前也是第 30 秒）
    → 幽灵会用轮廓线 + 淡色显示，不干扰当前游戏画面
    → 玩家可实时看到"过去的自己"在相同时间点的选择与位置
```

### 核心设计要素

**传承之魂的挑选**
- 玩家存档最多保留 3 条传承之魂，可手动命名（如"斧头流最高分"、"剑刀通关路线"）
- 结算界面显示该局关键数据（存活时长、最终等级、最多击杀武器），辅助玩家决策是否覆盖

**幽灵的表现方式**
- 幽灵经过的位置留下短暂的"足迹"，帮助玩家追溯过去的走位思路
- 幽灵升级时，屏幕角落显示"[魂] 选择了：旋转剑 Lv2"，让当前玩家感受到策略对比
- 幽灵死亡时，出现"[传承之魂已消散]"提示；幽灵通关时，出现"[传承之魂已完成轮回]"

**竞速维度**
- 系统实时对比当前玩家与幽灵的：当前等级差、当前击杀数差、HP 余量差
- 当前玩家在某项超越幽灵时，画面出现简短提示"超越传承！"
- 全程超越幽灵并通关 → 解锁成就"超越轮回"，并提示是否用新局覆盖传承之魂

**好友传承（可选扩展）**
- 玩家可以导出自己的传承之魂为一串代码，分享给朋友
- 朋友导入后，可以跟着"他人的幽灵"游玩，实现异步的社交竞速体验

### 设计优缺点

| 优点 | 缺点 |
|------|------|
| 强烈的自我成长感——跟自己竞争而非跟 AI | 需要玩家先有"值得保存"的局才能体验 |
| 幽灵是玩家自己的策略，不存在"AI 太蠢"的问题 | 早期存档的幽灵随玩家进步而显得过弱，失去参考价值 |
| 竞速元素增加重复游玩动力 | 幽灵路径与当前局差异过大时，参考意义下降 |

## 方案 C：神谕试炼（AI 托管 + 赛后复盘）

### 设计理念

> 来源于 AFK Arena 的"托管"哲学与移动游戏的"AI 代打"文化——但本方案在此基础上加入**复盘与学习**，使自动游玩不只是"看戏"，而是成为**提升玩家自身水平的工具**。

### 玩家体验流程

```
游戏内任意时刻，玩家可打开"神谕"菜单
    → 选择"启动神谕试炼"
    → 游戏进入【神谕模式】：AI 接管玩家控制权，玩家变为纯观战者

【神谕模式中】：
    → 屏幕边框变为金色，左上角显示"神谕试炼中"
    → AI 控制角色正常游玩，玩家全程旁观
    → 玩家可随时按暂停恢复自己的控制权（神谕试炼中止，成绩保留）

【试炼结束后（AI 通关或死亡）】：
    → 进入"神谕复盘"界面（耗时约 30 秒）
    → 复盘以时间轴形式展示本局 3 个关键节点：
        - AI 如何在 Boss 前快速拉开距离
        - AI 在升级时优先选择了哪个组合
        - AI 在 HP 危险时选择了哪个走位方向
    → 结束后显示："神谕本局评分：★★★（存活 18 分 40 秒）"
    → 玩家可选择：【再次托管】或【自己重试同一关】
```

### 核心设计要素

**神谕模式的解锁条件**
- 玩家第一次被 Boss 击败后，解锁神谕功能，并收到提示："你可以让神谕替你完成这一局，然后观察它的策略。"
- 强调这是学习工具，而非"开挂"

**复盘的内容深度**
- 复盘只展示 3 个节点，不淹没玩家（宁少勿多）
- 每个节点用两格对比展示："神谕的选择"vs"你上局的选择（如有记录）"
- 节点以玩家能理解的语言描述，如"神谕拉开了距离，避免被 Boss 的旋转攻击命中"

**托管结果与成就系统**
- 神谕通关不记入玩家的排行榜正式成绩，标注为"神谕代理局"
- 但神谕局的完成会解锁以下内容：
  - 观看神谕通关一次 → 解锁成就"旁观神明"
  - 在神谕复盘后，自己完成通关 → 解锁成就"青出于蓝"
  - 自己的最终分数超过神谕 → 解锁成就"超越神谕"，并获得专属称号

**神谕 AI 的性格化设计**
- 神谕 AI 并非追求满分的完美机器，而是有明显的"打法风格"：
  - **速攻流**：优先解锁伤害技能，激进靠近敌群收割
  - **龟壳流**：优先叠加防御与回血，稳步推进
  - **博彩流**：优先追求罕见武器组合，容易暴毙也容易爆发
- 玩家在解锁神谕时可以选择神谕的"性格"，看不同风格的打法，而非只有一套标准答案
- 每种性格的神谕有专属旁白台词，在关键节点出现文字提示（如速攻流："敌人太多？冲进去才能显出身手。"）

### 设计优缺点

| 优点 | 缺点 |
|------|------|
| 复盘系统将"看戏"转化为学习行为，体验层次最丰富 | 复盘界面设计复杂，开发成本最高 |
| 三种 AI 性格提供差异化的观战内容，重复观看仍有价值 | 玩家可能依赖神谕代打，削弱自主游玩动力 |
| "青出于蓝"成就链直接驱动玩家在观后尝试自己复现 | AI 性格描述可能与实际表现不符，产生落差 |

## 三方案横向对比

| 维度 | 方案 A：轮回见证者 | 方案 B：传承之魂 | 方案 C：神谕试炼 |
|------|------------------|----------------|----------------|
| **核心体验** | 旁观 + 随时接管 | 与自己的幽灵竞速 | 观战 + 复盘学习 |
| **AI 扮演的角色** | 游戏代言人 | 不使用 AI（回放自己） | 导师/风格展示者 |
| **触发时机** | 标题界面空闲时 | 游玩中（叠加层） | 游玩中（接管控制） |
| **奖励机制** | 接管成就 | 超越幽灵成就 | 复盘后通关成就 |
| **重复游玩动力** | 低（看腻了就不看） | 高（每局都能刷新记录） | 中（三种风格各看一遍后降低） |
| **新玩家友好度** | ★★★★★ | ★★★ | ★★★★ |
| **老玩家吸引力** | ★★ | ★★★★★ | ★★★ |
| **适合组合使用** | 可作为其他方案的"门面" | 可独立运行 | 可与方案 A 结合（标题演示用神谕 AI） |

---

## 推荐路线

**最快出效果**：方案 A（标题吸引模式）
- 对现有游戏循环改动最小，仅需 AI 在标题界面自动游玩
- 视觉冲击力强，是最好的"第一印象"功能

**长期留存价值最高**：方案 B（传承之魂）
- 与玩家个人数据深度绑定，每个人的体验都是唯一的
- 竞速机制驱动重复游玩，生命周期长

**体验最完整但开发量最大**：方案 C（神谕试炼）
- 如果目标是让玩家"因为 AI 而变得更强"，这是唯一做到这一点的方案
- 建议作为后期版本的旗舰功能
