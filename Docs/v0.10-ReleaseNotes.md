# v0.10 发布说明（Feature-编辑器制作）

## 核心亮点

- **数值与节奏更易调**：敌人波次、技能属性等关键参数支持通过 CSV 外置调整，减少改代码频率。
- **Boss 可扩展**：MiniBoss 重构为“类型 + 技能组合 + 状态机”，便于后续继续扩展更多 Boss 行为。
- **战斗体验更顺滑**：清怪后更快推进波次；修复部分技能对 Boss 无效等问题。
- **体验增强**：掉落物屏外引导（可开关）、技能图标 Tooltip（PC 悬停 / 移动端长按）。
- **多语言初步成型**：支持中英热切换；提供导出/导入工作流与冲突检测，覆盖生产链路。

## 玩法/体验变化

### 波次推进与混合波

- 波次配置支持“波次组/段落”的概念：同一组可以包含小怪段落与 Boss 段落。
- 清怪后更快切波，减少“清完怪还要等待下一段刷出”的干等体验。
- 提供刷新结束后的等待上限等兜底，避免拖波。

相关说明：`Assets/Docs/WaveSystem.md`、`Docs/GameConfig-Guide.md`。

### Boss（MiniBoss）

- Boss 支持多种类型（冲刺/弹幕/召唤/狂战/混合），每种类型对应不同技能组合。
- 行为由状态机驱动：追踪玩家 ↔ 执行技能。

### 掉落物引导（可开关）

- 屏外箭头指向掉落物；掉落时触发脉冲环与 3D 提示音。
- 设置项持久化保存（下次启动仍生效）。

### 技能图标 Tooltip

- PC：鼠标悬停显示技能名称/说明。
- 触屏：长按显示技能名称/说明。

## 配置与工具链变化（面向策划/数值/运营）

### 数值配置（CSV 生效路径）

- **运行时生效的配置**以 `Assets/StreamingAssets` 下的 CSV 为准（无需重新打包即可替换文件）。
- 敌人波次：`Assets/StreamingAssets/Config/EnemyWaveConfig.csv`
- 技能属性：`Assets/StreamingAssets/Config/AbilityConfig_i18n.csv`（优先），回退 `AbilityConfig.csv`

### 敌人属性配置（ScriptableObject）

- `EnemyStatsConfig` 统一管理敌人基础 HP/速度/伤害倍率/掉落等。
- 提供自定义 Inspector：支持自动填充预制体、批量编辑、搜索统计、CSV 导入导出。

### 校验与编辑器辅助

- 提供“CSV+Prefab”一致性校验菜单：用于检查 EnemyPrefabName 映射与预制体组件完整性。

## 多语言（i18n）

- 运行时表：`Assets/StreamingAssets/Localization/{table}.{lang}.csv`（`key,value`）
- `manifest.json` 描述支持语言与表清单。
- 支持中英热切换；UI 可通过 LocalizedText/LocalizedTMPText 自动刷新。
- 提供编辑器工具：合并导出为单表 CSV、再导入写回各语言表；包含 sha256 冲突检测，避免覆盖他人改动。

完整说明：`Docs/I18N.md`。

## 已知注意事项/回归点

- 资源改名（Enemy/Sprite 等）后，若出现引用丢失，优先检查 `.meta` 与 GUID 是否一致。
- 表驱动字段扩展后，旧 CSV 可能存在列缺失，需要按指南补齐或依赖默认值兜底。
- 多语言表请使用 UTF-8 编码保存；建议统一通过 Workbench 导出/导入流程协作。
