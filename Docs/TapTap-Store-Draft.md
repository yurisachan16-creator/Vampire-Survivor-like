## 简体中文长简介

`{正式中文名}` 是一款 2D 幸存者动作游戏。你将在不断增强的怪潮中移动、拾取、升级与进化武器，用越来越完整的构筑撑过整场战斗。

### 核心特色

1. 自动攻击 + 走位生存：专注移动与路线判断，在高密度敌群中打出节奏。
2. 升级三选一：每次升级都要在武器、属性和成长方向之间做取舍。
3. 宝箱与进化：满足条件后可将基础武器升级为更强形态，形成一轮爆发式质变。
4. 永久成长：金币可用于长期强化，配合成就系统提供持续解锁目标。
5. 多语言基础支持：项目已经具备简中、英文、日文、韩文与繁中的本地化能力。

### 当前内容

- 波次推进与 Boss 压力测试
- 多种武器、被动与掉落物组合
- 宝箱合成、成就、永久升级与本地排行榜
- 触屏操作支持，适配 Android 游玩
