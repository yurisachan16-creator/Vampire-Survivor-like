# TapTap 发布资产目录

本目录用于存放可直接提交到 TapTap 的最终素材与公开资料。

## 目录结构

- `Icon/`：安卓图标、商店图标、渠道图标
- `Hero/`：横版主视觉、头图
- `Capsule/`：竖版封面、胶囊图
- `Screenshots/`：真实游戏截图
- `Video/`：宣传视频封面、视频文件或上传清单
- `Policies/`：隐私政策公开页源文件、链接记录
