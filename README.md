 # Vampire Survivor-like

## 游戏结构

### 规则
- 通关条件：敌人最后一波且死亡。

### 数值
- 经验值
- 等级
- 时间
- 金币（永久升级用途）

---
## 主角能力升级&解锁
系统：主角能力（升级选项）
- 简单能力：每1.5秒攻击敌人，后续支持伤害提升与附加特效（文字飘动、伤害数字）
- 斧头：围绕主角旋转的固定物体，提升攻击范围和持续输出
- 小刀：从主角发射的直线飞行物体，适合远程打击
- 轰炸道具（如炸弹）：主动技能，爆炸时触发屏幕震动特效

---
## 系统
- 主角能力升级与解锁
- 永久道具与能力储存（金币/经验）
- 金币升级系统支持存档与等级限制
- QFramework 架构支持：BindableProperty、ViewController、AbstractSystem

---
## 波次系统
系统：
- 敌人（根据时间生成不同波次的敌人）
	- 三角形敌人：基础类型，向玩家移动或沿固定路径巡航，生命值/伤害平衡
	- 幽灵：可穿越障碍、移动速度快、伤害较低但扰乱节奏
	- 蝙蝠：数量多、体力较低、擅长群体骚扰
	- 精英敌人：高生命、高伤害、具备特殊能力（远程攻击、分裂、回血等）

### 流程（循环）
- 游戏中
- 游戏结束
- 游戏通关

#### 流程图
- 游戏中 → 敌人全部死亡 → 游戏通关 → 再来一局 → 游戏中
- 游戏中 → 主角死亡 → 游戏结束 → 再来一局 → 游戏中

---
## 掉落
- 经验值：敌人被击败后掉落，推动等级提升与升级面板触发
- 金币：敌人掉落，用于永久升级和开启新能力
- 杀掉所有敌人：一次性获得所有经验值奖励
- 恢复HP：掉落回血道具，拾取后为主角恢复生命值
- 炸弹/重击：击中时可触发屏幕震动，带来视觉反馈

---
## 永久升级
系统：
- 提高经验值的掉落概率
- 增加金币掉落概率
- 主角最大血量+1
- 带存档的金币升级界面，支持等级提升次数限制与界面优化
