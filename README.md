# Vampire Survivor-like

## 游戏结构

### 规则
- 通关条件：敌人最后一波且死亡。

### 数值
- 经验值
- 等级
- 时间
- 金币（永久升级用）

### 系统
- 主角能力（升级选项）
	- 简单能力：每1.5秒攻击敌人

### 流程（循环）
- 游戏中
- 游戏结束
- 游戏通关

#### 流程图
- 游戏中 → 敌人全部死亡 → 游戏通关 → 再来一局 → 游戏中
- 游戏中 → 主角死亡 → 游戏结束 → 再来一局 → 游戏中

---

## 永久升级
系统：
- 提高经验值的掉落概率
- 金币掉落概率

## 波次系统
系统：
- 敌人（根据时间生成不同波次的敌人）
	- 三角敌人

## 敌人的扩展
- 三角形敌人：基础敌人类型，行为可定义为简单向玩家移动或沿固定路径移动，具备基础生命值与伤害。
- 扩展说明：后续可添加更多敌人类型（例如快速型、坦克型、远程型），用于丰富不同波次的挑战。

## 掉落
- 经验值：敌人被击败后掉落经验值，用于提升主角等级和解锁/强化能力。
- 金币：敌人掉落金币，用于永久升级（例如提升经验掉落概率、金币掉落概率等）。
